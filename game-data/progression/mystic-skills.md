# Where Winds Meet - Mystic Skills (奇术)

## Mystic Skills System Overview

**Total Count**: 23 Mystic Skills

**Equipment Limit**: Up to 8 Mystic Skills can be equipped simultaneously

**Independence**: NOT tied to weapons - can use with any weapon setup

**Dual Function**: Useful in both combat AND exploration/puzzle-solving

---

**Type**: Mobility/Movement

**Functions**:

**Functions**:

## Mystic Skill Loadout Strategy

### 8-Slot System
Players must choose 8 from 23 available skills

**Strategy Considerations**:
- Balance combat and exploration skills
- Match skills to current content
- Build synergies between skills
- Situational swapping

### Recommended Loadout Structure
- **3-4 Combat Skills**: Direct damage or crowd control
- **2-3 Utility Skills**: Mobility and positioning
- **1-2 Exploration Skills**: Puzzle-solving and traversal
- **Flexibility**: Adapt based on activity

## Mystic Skill Progression

### Unlocking Mystic Skills
- **Story Progression**: Unlock through main quests
- **Side Quests**: Discover through side content
- **Exploration**: Find in hidden locations
- **Masters**: Learn from NPCs
- **Achievements**: Unlock through accomplishments

### Upgrading Mystic Skills
- **Skill Levels**: Increase power and effectiveness
- **Enhanced Effects**: Better damage, longer duration, shorter cooldown
- **Chi Efficiency**: Reduce Chi cost at higher levels
